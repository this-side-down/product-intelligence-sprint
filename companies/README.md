# Companies

One folder per company sprint.

## Convention

```
companies/
  <company-slug>/
    sprint.md          # primary sprint document (from templates/sprint-template.md)
    notes/             # optional — supplementary memos, interview-adjacent field notes, etc.
```

Pair each company folder with a matching `artifacts/<company-slug>/` directory for source material.

Do not start a company folder until a sprint is actively queued or in progress.
